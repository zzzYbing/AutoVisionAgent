# AutoVisionAgent

一个基于多 Agent 协作的工业视觉检测自动优化系统，用于解决传统视觉算法中参数依赖人工调试、稳定性差的问题。

---

## 📌 项目背景

在工业视觉检测（如孔位检测、圆拟合）中，通常依赖人工反复调整如下参数：

- 二值化阈值（Threshold）
- 滤波参数（Gaussian Blur）
- 边缘检测参数（Canny）
- 拟合策略

在实际产线环境中，由于光照变化、工件差异等因素，这些参数难以复用，导致：

❗ 调试耗时长（单次可达30分钟）  
❗ 检测结果不稳定  
❗ 对工程师经验依赖强  

---

## 🎯 项目目标（核心痛点）

本项目构建一个**AI驱动的自动优化 Agent 系统**，解决以下问题：

- 自动完成视觉参数调优（替代人工）
- 提升检测稳定性与鲁棒性
- 降低对经验依赖
- 构建可扩展的自动优化框架

---

## 🧠 核心架构（多 Agent 协作）

系统采用多 Agent 解耦设计，形成闭环优化流程：


图像输入
↓
[ImageAnalyzer] → 图像感知 + 参数生成
↓
[Detector] → 执行检测（边缘提取 + 圆拟合）
↓
[Evaluator] → 结果评分（误差 + 点数）
↓
参数优化 / 收敛判断
↓
输出最优结果


### 🔹 1. ImageAnalyzer（感知 Agent）
- 分析图像亮度分布
- 生成候选参数组合
- 缩小搜索空间（避免暴力穷举）

### 🔹 2. Detector（执行 Agent）
- 图像预处理（滤波、二值化）
- 边缘检测（Canny）
- 点集提取
- 最小二乘圆拟合

### 🔹 3. Evaluator（评估 Agent）
- 基于拟合误差与点数量构建评分函数
- 自动判断结果优劣

---

## 🔁 核心逻辑流（长链推理）

本系统实现了类似“长链推理”的自动优化流程：


参数生成 → 执行检测 → 结果评估 → 参数更新 → 收敛判断


特点：

- 自动搜索最优参数组合
- 基于反馈不断优化
- 引入收敛机制避免无效迭代
